## 项目简介

本项目实现了多智能体机器人足球仿真，基于 ROS2 和 Isaac Sim 平台。支持多台机器人之间的协作与对抗，可用于研究和实验多智能体系统的算法与通讯。

## 基本功能

- 多智能体协作与对抗仿真
- 基于 ROS2 的消息通信
- 可以扩展和集成新的智能体算法

## 快速开始

1. 克隆本仓库
2. 按照 Makefile 或 CMake 文件中的说明完成依赖安装
3. 运行 ROS2 相关指令启动仿真环境

## 依赖要求

- ROS2
- Isaac Sim
- C++ 编译环境
- Python 3
- 其他依赖请参阅源码及构建脚本
